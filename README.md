### [_ca-dot-rtmp_](https://github.com/warren-bank/node-list-all-working-CA-DOT-RTMP-traffic-cams)

Generate a list of working RTMP traffic cam streams from the CA DOT.

#### Installation

```bash
npm install --global "@warren-bank/ca-dot-rtmp"
```

#### Usage

```bash
ca-dot-rtmp ["/path/to/output.txt"] [region-id]
```

* `["/path/to/output.txt"]`
  - default: `./ca-dot-rtmp.txt`
* `[region-id]`
  - type: integer
  - note: region id values that are currently associated with traffic cams are in the range: `1` &hellip; `12`
  - default: include all region ids in the range: `1` &hellip; `20`
* [_sample output_](./test/ca-dot-rtmp.txt)
  - generated by:
    ```bash
      npm run test
    ```
