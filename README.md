### [_ca-dot-rtmp_](https://github.com/warren-bank/node-list-all-working-CA-DOT-RTMP-traffic-cams)

Generate a list of working RTMP traffic cam streams from the CA DOT.

#### Installation

```bash
npm install --global "@warren-bank/ca-dot-rtmp"
```

#### Usage

```bash
ca-dot-rtmp ["/path/to/output.txt"]
```

* `["/path/to/output.txt"]`
  - default: `./ca-dot-rtmp.txt`
